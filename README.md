JPush Docs
==========

## 目录

*  安装 virtualenv
```
pip install virtualenv
```
*  在 /opt/push/jpush-docs 创建虚拟环境
```
virtualenv venv
```
*  激活虚拟环境
```
. venv/bin/activate
```
## 依赖
```
pip install requests
```
```
pip install mkdocs
```
## mkdocs 官方文档
http://www.mkdocs.org/

## 运行
```
python autobuild.py
```
