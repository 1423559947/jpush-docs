# IM SDK for iOS

### Summary 概述

JPush IM SDK 基于 JPush 推送 SDK 开发，提供了 Push SDK 的完整功能，并提供 IM 即时通讯功能。

App 集成了 IM SDK 就不应再集成 Push SDK（只提供 Push 功能的 SDK）。
